# Simple Weather App

A simple weather application that fetches real-time weather data using an API. 

# Features
- Displays current weather conditions.
- Fetches temperature, humidity, and wind speed.
- Simple and responsive UI.

# Technologies Used
- HTML, CSS, JavaScript
- OpenWeather API
- Bootstrap (Optional)
